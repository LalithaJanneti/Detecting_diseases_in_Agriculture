# Detecting_diseases_in_Agriculture
1.🥔 Potato Disease Classification using Deep Learning

## 📌 Problem Statement

Potato is one of the most widely cultivated crops and a major source of income for farmers. However, potato plants are highly vulnerable to diseases such as:

* 🍂 **Early Blight**
* 🌧️ **Late Blight**
* 🌿 **Healthy Leaves (Normal condition)**

These diseases can significantly reduce crop yield and quality if not identified early.

### ❗ Challenges Faced by Farmers

* Lack of expert knowledge to identify diseases early
* Difficulty in distinguishing between similar-looking leaf conditions
* Delay in treatment leading to crop loss
* Limited access to agricultural specialists in rural areas

## 💡 Proposed Solution

This project provides an **AI-powered solution** to help farmers quickly detect potato leaf diseases using images.

👉 By simply uploading an image of a potato leaf, the system can predict whether the plant is:

* ✅ Healthy
* ⚠️ Affected by Early Blight
* 🚨 Affected by Late Blight

## 🧠 How It Works

1. 📷 User uploads a potato leaf image
2. 🔄 Image is preprocessed (resizing, normalization)
3. 🤖 A trained **Deep Learning CNN model** analyzes the image
4. 📊 The model classifies the image into one of the disease categories
5. 📢 Output is displayed with prediction

## ⚙️ Tech Stack

* 🐍 Python
* 🧠 TensorFlow / Keras
* 🔢 NumPy, Pandas
* 📊 Matplotlib
  

## 📂 Dataset

The model is trained on a dataset containing labeled images of:

* Potato Early Blight
* Potato Late Blight
* Healthy Potato Leaves
  
## 📈 Model Performance

* Achieved high accuracy in classifying potato leaf diseases
* Capable of generalizing well to unseen images

## 🌱 Impact on Farmers

✔ Helps farmers take timely action
✔ Supports precision agriculture
